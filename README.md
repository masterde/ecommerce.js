# What is Ecommerce.js?
The proposal of Ecommerce.js is create a good Ecommerce Platform building with modern tools like Node.js.

The Idea behind is simple, creating a Ecommerce Platform under these pillars:

- Extensible
- Extreme Pluggable
- Easy Maintenance
- Low cost resources
